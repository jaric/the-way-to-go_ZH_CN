# 11.2 接口嵌套接口

一個接口可以包含一個或多個其他的接口，這相當於直接將這些內嵌接口的方法列舉在外層接口中一樣。

比如接口 `File` 包含了 `ReadWrite` 和 `Lock` 的所有方法，它還額外有一個 `Close()` 方法。

```go
type ReadWrite interface {
    Read(b Buffer) bool
    Write(b Buffer) bool
}

type Lock interface {
    Lock()
    Unlock()
}

type File interface {
    ReadWrite
    Lock
    Close()
}
```
