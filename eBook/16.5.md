# 16.5 不需要將一個指向切片的指針傳遞給函數

在[第4.9小節](04.9.md)，我們已經知道，切片實際是一個指向潛在數組的指針。我們常常需要把切片作爲一個參數傳遞給函數是因爲：實際就是傳遞一個指向變量的指針，在函數內可以改變這個變量，而不是傳遞數據的拷貝。

因此應該這樣做：

        `func findBiggest( listOfNumbers []int ) int {}`

而不是：

       `func findBiggest( listOfNumbers *[]int ) int {}` 

**當切片作爲參數傳遞時，切記不要解引用切片。**
